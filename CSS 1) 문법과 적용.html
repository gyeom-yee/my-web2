h1 {
color: yellow;
font-size:2em;
}

선택자(selector) - "h1"
속성(property) - "color" / html의 속성은 Attribute
값(value) - "yellow"
선언(declaration) - "color: yellow", "font-size: 2em"
선언부(declaration block) - "{ color: yellow; font-size:2em; }"
규칙(rule set) - "h1 { color: yellow; font-size:2em; }"

내용


HTML
HTML은 문서의 구조를 나타냅니다.
CSS
CSS는 문서의 표현을 나타냅니다.
JS
JS는 문서의 동작을 나타냅니다.

id 선택자는 해쉬# 기호를 사용합니다.

  1. 기호가 아닌 #기호 사용
  2. 태그의 class 속성이 아닌 id 속성을 참조
  3. 문서 내에 유일한 요소에 사용
  4. 구체성
