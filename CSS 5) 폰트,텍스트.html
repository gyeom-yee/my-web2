< 폰트 속성 >

1. font-family 속성
font-family: family-name | generic-family ( | initial | inherit );

2. line-height 속성
line-height: normal | number | length | initial | inherit ;
예시) body { font-size: 20px; line-height: 2; }
* 부모 요소의 숫자 값이 그대로 상속되게끔 절대값을 쓰는게 좋음

3. font-size 속성
font-size: keyword | length | initial | inherit ;
예시) .parent {font-size: 20px;} .child {font-size: 1em;}

4. font-weight 속성
font-weight: normal | bold | bolder | lighter | number | initial | inherit ;

5. font-style 속성
font-style: normal | italic | oblique | initial | inherit;

6. font-variant 속성
Font-Variant: Small-Caps font-variant: normal | small-caps | initial | inherit ;

7. font 축약형 속성(실무에선 지양)
font: font-style font-variant font-weight font-size/line-height font-family | initial | inherit;

8. @font-face 속성
웹에 있는 글꼴을 사용자의 로컬 환경(컴퓨터)으로 다운로드하여 적용하는 속성
  • family(필수): 글꼴의 이름을 지정(그냥 명시 목적)
  • src(필수): 다운로드 받을 글꼴의 경로(URL)
  • font-style(옵션): 글꼴의 스타일 지정, 기본 값은 normal
  • font-weight(옵션): 글꼴의 굵기 지정, 기본 값은 normal

9. vertical-align 속성
vertical-align: keyword | length | percent | initial | inherit ;
* 음수는 아래로, 인라인라벨에만 선언가능

10. text-align 속성
text-align: left | right | center | justify | initial | inherit ;
    ※ text-align과 display의 관계
  • text-align은 inline-level에 적용
  • 블록요소 정렬방법: 박스모델 챕터에서 다룬 margin의 auto 값을 이용
  • 가운데 정렬 인라인 요소 : text-align (center) 블럭 요소 : margin (auto)
  • 요소의 레벨에 따라 정렬하는 방식

11. text-indent 속성
text-indent: length | initial | inherit;

12. text-decoration 속성
text-decoration: text-decoration-line text-decoration-color text-decoration-style | initial | inherit;
none | underline | overline | line-through

13. white-space 속성
white-space: normal | nowrap | pre | pre-line | pre-wrap | initial | inherit;
    < 속성 값 >
  • normal: 공백, 개행X / 필요한 경우 자동 줄바꿈 O 기본 값
  • nowrap: 공백, 개행X / 자동 줄바꿈 X
  • pre: 공백, 개행 O / 자동 줄바꿈 X
  • pre-wrap: 공백O, 개행X / 필요한 경우 자동 줄바꿈 O

14. letter-spacing 속성
(자간 지정)
letter-spacing: normal | length | initial | inherit;

15. word-spacing 속성
word-spacing: normal | length | initial | inherit;

16. word-break 속성
word-break: normal | break-all | keep-all | initial | inherit;
단어가 라인 끝에 나올 경우 어떻게 처리할지(중단점) 지정하는 속성

17. word-wrap 속성
word-wrap: normal|break-word|initial|inherit;
요소를 벗어난 단어의 줄바꿈을 지정하는 속성
